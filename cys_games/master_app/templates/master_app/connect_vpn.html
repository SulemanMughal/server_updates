How to connect VPN

  1. Home
  2. How to connect VPN

Log in to Cyber Range and on top right side, you'll see the "Create VPN" Link.

Create your own individual .ovpn file. Once you're able to create .ovpn file, it will be available to download.

Download the .ovpn file

For accessing the vpn, go to the machine and install openvpn.

Now type openvpn filename.ovpn

You'll get the status "initialization sequnce completed"

Now you're connected and can access whatever machine you want
